# 🐍 贪吃蛇小游戏

一个使用纯HTML、CSS和JavaScript创建的经典贪吃蛇游戏。

## 🎮 游戏特点

1. **经典玩法** - 传统的贪吃蛇游戏规则
2. **现代界面** - 美观的渐变背景和UI设计
3. **响应式设计** - 适配各种屏幕尺寸
4. **难度选择** - 三种难度级别可选
5. **本地存储** - 自动保存最高分记录
6. **键盘/按钮控制** - 支持键盘和鼠标操作

## 📁 文件结构

```
snake_game/
```

## 🕹️ 游戏控制

### 键盘控制
- **方向键** (↑ ↓ ← →) - 控制蛇的移动方向
- **空格键** - 开始/暂停游戏
- **P键** - 暂停游戏

### 按钮控制
- **开始游戏** - 开始游戏
- **暂停游戏** - 暂停游戏
- **重新开始** - 重新开始游戏

## 🎯 游戏规则

1. 使用方向键控制蛇的移动方向
2. 吃到红色食物可以增加长度和得分（每吃一个得10分）
3. 撞到墙壁或自己的身体游戏结束
4. 游戏难度越高，蛇移动速度越快
5. 尽可能获得高分，挑战自己的极限！

## ⚙️ 难度级别

- **简单** - 蛇移动速度较慢，适合新手
- **普通** - 标准速度，适合大多数玩家
- **困难** - 蛇移动速度很快，挑战反应极限

## 💾 功能特性

- **最高分记录** - 使用localStorage保存最高分
- **游戏状态显示** - 实时显示游戏状态（游戏中/暂停/结束）
- **蛇身渐变效果** - 蛇身颜色从头部到尾部渐变
- **食物动画** - 食物有高光效果，更像苹果
- **蛇眼睛** - 蛇头有眼睛，会根据移动方向变化

## 🚀 快速开始

1. 直接在浏览器中打开 `index.html` 文件
2. 无需任何服务器或网络连接
3. 点击"开始游戏"按钮或按空格键开始游戏

## 🔧 自定义修改

你可以轻松修改以下参数来自定义游戏：

1. **游戏速度** - 修改`gameSpeed`变量
2. **网格大小** - 修改`GRID_SIZE`常量
3. **画布尺寸** - 修改canvas的width和height属性
4. **颜色主题** - 修改CSS中的颜色值
