# 我的小网页

这是一个使用纯HTML、CSS和JavaScript创建的响应式网页示例。

## 功能特点

1. **响应式设计** - 自适应各种屏幕尺寸
2. **现代UI** - 使用CSS渐变、阴影和过渡效果
3. **交互功能** - 包含JavaScript交互式计数器
4. **键盘支持** - 支持键盘操作计数器

## 文件结构

```
webpage/
├── index.html      # 主网页文件
└── README.md       # 项目说明文档
```

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 无需任何服务器或网络连接
3. 支持所有现代浏览器

## 技术栈

- **HTML5** - 语义化标签结构
- **CSS3** - Flexbox、Grid布局、渐变、过渡
- **JavaScript** - DOM操作、事件处理

## 交互功能

- 点击按钮增加/减少/重置计数器
- 键盘支持：`+`/`=` 增加，`-`/`_` 减少，`0` 重置
- 计数器颜色根据数值变化（正数绿色，负数红色，零蓝色）

## 设计特点

- 现代化渐变背景
- 卡片式布局
- 平滑的过渡动画
- 移动端友好的响应式设计

## 浏览器兼容性

支持所有现代浏览器（Chrome、Firefox、Safari、Edge等）

## 许可证

本项目仅供学习参考，可自由使用和修改。
