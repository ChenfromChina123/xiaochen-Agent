# Python 贪吃蛇游戏

这是一个使用 Pygame 库实现的经典贪吃蛇游戏，具有完整的图形界面和丰富的游戏功能。

## 功能特点

- 🎮 完整的图形界面
- 🐍 精美的蛇和食物绘制（包含特殊食物）
- ⚡ 三种难度级别（简单、普通、困难）
- 📊 分数系统和最高分记录
- 🎯 游戏状态管理（开始、暂停、结束）
- 🖱️ 鼠标和键盘双重控制
- ✨ 特殊食物系统（金色食物，加分更多且有特殊效果）
- 🎨 美观的UI设计和动画效果

## 运行要求

### 系统要求
- Python 3.6 或更高版本
- Windows、macOS 或 Linux 系统

### 依赖库
- Pygame 2.0 或更高版本

## 安装和运行

### 1. 安装 Python
确保你的系统已安装 Python 3.6 或更高版本。

### 2. 安装 Pygame
打开终端或命令提示符，运行以下命令安装 Pygame：

```bash
pip install pygame
```

### 3. 运行游戏
进入游戏目录，运行以下命令：

```bash
cd python_snake_game
python snake_game.py
```

或者直接运行：

```bash
python python_snake_game/snake_game.py
```

## 游戏控制

### 键盘控制
- **方向键 (↑ ↓ ← →)** - 控制蛇的移动方向
- **空格键** - 开始/暂停游戏
- **P 键** - 暂停游戏
- **R 键** - 重新开始游戏
- **ESC 键** - 返回菜单/退出游戏

### 鼠标控制
- 点击屏幕上的按钮进行游戏操作
- 点击难度按钮选择游戏难度

## 游戏规则

1. 使用方向键控制蛇的移动方向
2. 吃到红色食物可以增加蛇的长度和得分
3. 撞到墙壁或自己的身体游戏结束
4. 游戏难度越高，蛇移动速度越快
5. 尽可能获得高分，挑战自己的极限！

## 游戏界面说明

### 主菜单
- 开始游戏按钮
- 难度选择（简单、普通、困难）
- 游戏规则说明

### 游戏界面
- **左侧**：游戏区域，显示蛇和食物
- **右侧**：游戏信息和控制面板
  - 当前得分和最高分
  - 蛇的当前长度
  - 游戏状态显示
  - 控制按钮
  - 难度选择按钮

### 游戏状态
- **游戏中**：绿色状态显示，蛇正常移动
- **暂停**：橙色状态显示，游戏暂停
- **游戏结束**：红色状态显示，显示最终得分

## 技术特点

1. **面向对象设计**：使用类封装游戏逻辑
2. **状态管理**：使用枚举管理游戏状态
3. **事件驱动**：同时支持键盘和鼠标事件
4. **图形渲染**：使用 Pygame 进行图形绘制
5. **数据持久化**：自动保存和加载最高分

## 自定义修改

你可以根据需要修改以下参数：

### 游戏参数
- 屏幕大小：修改 `screen_width` 和 `screen_height`
- 游戏区域：修改 `game_area_x`, `game_area_y`, `game_area_width`, `game_area_height`
- 网格大小：修改 `grid_size`
- 游戏速度：修改 `game_speeds` 字典中的值

### 颜色主题
在 `colors` 字典中修改颜色值，可以自定义游戏主题。

### 难度设置
在 `game_speeds` 字典中调整各难度的移动间隔时间（毫秒）。

## 常见问题

### 1. 运行游戏时出现 "ModuleNotFoundError: No module named 'pygame'"
请确保已正确安装 Pygame：
```bash
pip install pygame
```

### 2. 游戏运行缓慢或卡顿
- 降低游戏难度
- 关闭其他占用系统资源的程序
- 确保显卡驱动已更新

### 3. 最高分无法保存
确保游戏目录有写入权限，或者手动创建 `high_score.txt` 文件。

## 开发说明

### 代码结构
- `SnakeGame` 类：游戏主类
- `Direction` 枚举：方向定义
- `Difficulty` 枚举：难度定义
- `GameState` 枚举：游戏状态定义

### 主要方法
- `__init__()`：初始化游戏
- `run()`：游戏主循环
- `handle_events()`：处理用户输入
- `update_game()`：更新游戏逻辑
- `draw_*()`：各种绘制方法
